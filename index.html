Become a Collaborator

Step #1

Introduce yourself and get approved as a collaborator.

Step #2

  • In your terminal, visit your desktop
    cd ~/Desktop
  • In your terminal, type:
    git clone https://github.com/iarobinson/learn-to-be-a-useful-programmer.git
  • Then move into the folder you just created:
    cd learn-to-be-a-useful-programmer
  • Create a new git branch with your name
    git checkout -b YOUR_NAME
  • Open `index.html` in your text editor
  • Add your name to the table using HTML (replace YOUR NAME HERE with your name in the section below)
  • Save your changes to the index.html file (cmd + s)
  • In your terminal, run the command `git add .` to stage your changes
  • In your terminal, run the command `git commit -m "Add my first commit"` to commit with a message
  • In your terminal, run the command `git push`
  • In your terminal, you'll see an error with a message that tells you to set upstream master
  • Run that command, for me it was:
    git push --set-upstream origin iarobinson/update_instructions
  • Navigate to Pull Request tab on GitHub using a web browswer like Chrome or Safari
  • You should be able to click an icon which will create a pull request from the branch you created
  • Create a new pull request
  • Once your pull request has been created, tweet me and we can figure out the next thing for you to work on. :D
  • Celebrate!
